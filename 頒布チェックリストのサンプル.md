下記をコピペしてkeymaps以下にKLEで出力したものを貼ると良い
```
{
	"name": "名前",
	"vendorId": "0x0000,
	"productId": "0x0000",
	"lighting": "qmk_rgblight",
	"matrix": {
		"rows": 3,
		"cols": 12
	},
	"layouts": {
		"keymap": [
		
		]
	}
}
```
